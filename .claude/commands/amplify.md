You are kicking off an Amplify session — finding where AI can make the user dramatically more effective in their CURRENT job.

## Pre-Flight

1. Load context if it exists (helpful, not blocking):
   - `profile/profile.md` — pass contents to the agent; do not re-ask what it answers
   - `profile/skills.md` — reveals technical fluency and learning capacity
   - If neither exists, proceed — the intake covers the current role directly. Suggest `/consult` afterward to unlock the rest of the harness.

2. Check for an existing plan: `profile/amplify-plan.md`
   - If it exists, ask: "You have an Amplify Plan already — review progress and expand it, or start fresh?"
   - Progress review: walk the Measurement Log, celebrate wins, convert measured results into resume bullet drafts, then add next plays.

## Execution

Invoke the **Amplify Strategist** agent to run the full session:

1. **Intake** — current role, responsibilities, recurring tasks with hours, tools/stack, pain points, success metrics, company AI policy
2. **Task decomposition** — build the task inventory; classify each task automate / augment / human-core
3. **Opportunity mapping** — concrete AI play per task; WebSearch to verify current tool options for the user's stack; score on impact × effort
4. **Plan delivery** — quick wins (this week), quarter plays (90 days), strategic positioning, learning path, measurement plan, guardrails
5. **Career feedback loop** — baseline every play so wins become quantified resume bullets; connect to `/skill-inventory` and `/build-resume`

## Output

Save to: `profile/amplify-plan.md`

## Success Criteria

- [ ] Task inventory complete with hours and classification
- [ ] At least 3 quick wins with concrete first steps
- [ ] Every play has an expected gain and a way to measure it
- [ ] Human-core work identified and explicitly protected
- [ ] Guardrails section reflects the company's actual AI policy (or flags it as unknown)

## Next Steps

After the session, recommend:
> "Your Amplify Plan is ready. Next:
> 1. Start quick win #1 today — log your baseline first
> 2. Check back in 30 days: `/amplify` again to measure results and expand
> 3. Measured wins become resume bullets — run `/skill-inventory` after 60–90 days to capture your new AI skills"
